# Signal Lab

Laboratorio interactivo de Procesamiento de Señales
para la asignatura:

Sistemas Electrónicos y de Instrumentación Industrial

Máster en Ingeniería Industrial
Universidad de La Laguna (ULL)

---

## Objetivos

- Visualizar convolución y correlación
- Comprender sistemas LTI
- Introducir filtros y respuesta temporal
- Facilitar aprendizaje visual e intuitivo

---

## Tecnologías

- HTML5
- CSS3
- JavaScript Vanilla
- SVG
- MathJax

---

## Arquitectura

state.js         → estado global  
signals.js       → señales de entrada  
filters.js       → sistemas/filtros  
convolution.js   → operador matemático  
render.js        → visualización SVG  
app.js           → coordinación y eventos

---

## Ejecución

Abrir:

index.html

o usar Live Server en VS Code.
